Introduction

Welcome to the H2O Rideau documentation site. Here you'll find everything you need to know to use our intelligent document search system.

Usage

Install the package and start using the API:

npm install h2o-rideau

Import and call:

import { searchDocs } from 'h2o-rideau';

searchDocs("example query");

API Reference

GET /collections — Fetch available collections

POST /query/response — Submit query to specific collection

FAQ

Q: How do I integrate this into my site?
A: You can use the JavaScript SDK or REST API directly.
